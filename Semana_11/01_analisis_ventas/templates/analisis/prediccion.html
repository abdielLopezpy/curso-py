📚 Como leer estos datos

  • R² (Coeficiente de Determinacion): Indica que porcentaje de la variacion en ventas es explicada por el modelo. Un valor cercano a 1.0 indica buen ajuste.
  • Pendiente (Slope): Cuanto cambian las ventas por cada mes que pasa. Un valor positivo indica crecimiento.
  • Intercepto: Valor base de ventas cuando el tiempo es 0 (inicio del periodo).
  • Formula: Ventas = {{ prediccion.pendiente }} × Mes + {{ prediccion.intercepto }}
